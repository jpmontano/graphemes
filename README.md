# Graphemes

**Graphemes is a Python 3.4 program which splits a Nishnaabe word (Pic River dialect) into its constituent graphemes.**


## Sample Output

```python
>>> "nishnaabe"
['n', 'i', 'sh', 'n', 'aa', 'b', 'e']

>>> "boozhoo"
['b', 'oo', 'zh', 'oo']

>>> "aaniin"
['aa', 'n', 'ii', 'n']

>>> "waabndaan"
['w', 'aa', 'b', 'nd', 'aa', 'n']

>>> "en'goons"
['e', 'n', "'", 'g', 'oo', 'ns']

>>> "biiwaan'goonh"
['b', 'ii', 'w', 'aa', 'n', "'", 'g', 'oo', 'nh']

>>> "gnoon'diwag"
['g', 'n', 'oo', 'n', "'", 'd', 'i', 'w', 'a', 'g']

>>> "gaa-gii-ni-zhaang"
['g', 'aa', '-', 'g', 'ii', '-', 'n', 'i', '-', 'zh', 'aa', 'ng']

>>> "endgwen"
['e', 'nd', 'g', 'w', 'e', 'n']

>>> "gzaaghin"
['g', 'z', 'aa', 'g', 'h', 'i', 'n']

>>> "wzhashk"
['w', 'zh', 'a', 'shk']

>>> "wzhashkoons"
['w', 'zh', 'a', 'shk', 'oo', 'ns']

>>> "aanjtoon"
['aa', 'nj', 't', 'oo', 'n']

>>> "kiwenzii"
['k', 'i', 'w', 'e', 'nz', 'ii']

>>> aanzkon'ye
['aa', 'nz', 'k', 'o', 'n', "'", 'y', 'e']

>>> giigoonyke
['g', 'ii', 'g', 'oo', 'ny', 'k', 'e']
```
